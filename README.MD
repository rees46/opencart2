# REES46 Module for CMS OpenCart 2.3.x.x

## Compatibility

* OpenCart 2.3.x.x (for OpenCart 2.0.1.x - 2.2.x.x use this module https://github.com/rees46/opencart2/tree/master)
* Module Simple Checkout
